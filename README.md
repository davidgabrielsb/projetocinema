# Projeto Cinema
Projeto de modelagem de um sistema de reserva e venda de ingressos para uma rede de cinemas. O sistema permite consultar filmes e sessões, reservar e comprar ingressos (incluindo meia-entrada), aplicar regras de fidelidade, controlar a disponibilidade de assentos em tempo real e gerar relatórios obrigatórios integrados à ANCINE.
